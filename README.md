# 📱 Calculator v1

A basic Android calculator app built using Java and XML in Android Studio.  
This is **Version 1** of the project, focusing on core arithmetic operations with a minimal UI.

---

## ✨ Features

- ➕ Addition  
- ➖ Subtraction  
- ✖️ Multiplication  
- ➗ Division  
- % Modulus (Remainder)  
- 🧮 Clear input  
- ✅ Instant calculation on "="
